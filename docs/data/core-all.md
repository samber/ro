Determines whether all elements of an observable sequence satisfy a condition.

```go
obs := ro.Pipe(
    ro.Just(1, 2, 3, 4, 5),
    ro.All(func(i int) bool {
        return i > 0
    }),
)

sub := obs.Subscribe(ro.PrintObserver[bool]())
defer sub.Unsubscribe()

// Next: true
// Completed
```
